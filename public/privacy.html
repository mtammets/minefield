Privacy

This policy explains what data Minefield Drift processes and why. The game does not require user accounts, but some technical and payment-related data is still processed to run the service. Optional analytics data is processed only after cookie consent.

Data We Process

  • Session and gameplay data: player name, room code, and real-time multiplayer state needed to run online sessions.
  • Technical/security data: IP address, basic request metadata, and short-lived rate-limit counters to protect service availability.
  • Donation data: donation amount and checkout outcome. Card and wallet details are processed by Stripe, not by this game server.
  • Optional analytics data: page views and basic usage metrics collected via Google Analytics only when analytics cookies are accepted.

Why We Process Data

  • To run gameplay features (including multiplayer rooms and session flow).
  • To prevent abuse, fraud, and service disruption.
  • To create Stripe Checkout donation sessions and return status to the game.
  • To understand feature usage and improve game UX when consent is granted.

Storage and Retention

  • Active multiplayer room data is held in server memory during live sessions.
  • Security rate-limit entries are short-lived in-memory records and are periodically pruned.
  • Payment records are stored by Stripe under Stripe's own retention and legal obligations.

Third-Party Processing

Donations use Stripe Checkout. Stripe acts as payment processor and may process personal/payment data under its own terms and privacy policy: stripe.com/privacy.

Optional usage analytics are provided by Google Analytics: policies.google.com/privacy.

Cookies and Consent

  • Analytics cookies are optional and disabled by default until you accept them.
  • You can change your analytics cookie choice at any time using the controls below.

Your Rights

If you are in the EU/EEA, you may have rights to access, rectify, erase, restrict, or object to processing, and to lodge a complaint with your local data protection authority.

Contact

For privacy questions or requests, contact: support@minefield.games.
